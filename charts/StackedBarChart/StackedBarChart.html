Stacked Bar Charts: Fines by Detection Method

These two stacked bar graphs compare the number of fines issued per jurisdiction, broken down by detection method (camera-issued or police-issued). The visualization helps identify patterns in how different jurisdictions enforce mobile phone usage laws.

How to Use the Filters:

  • Age Groups: Filter the data to focus on specific age demographics
  • Detection Methods: Toggle between camera-issued and police-issued fines
  • Jurisdictions: Select specific states or territories to compare

Use the "Reset Filters" button to clear all selections and return to the default view.
