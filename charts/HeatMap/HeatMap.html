Heat Map: Monthly Fines Distribution

This heat map displays the total number of fines issued per jurisdiction, by month, throughout 2023. The intensity of the color indicates the number of fines issued, with darker shades representing higher numbers. This visualization helps identify seasonal patterns and compare enforcement levels across different jurisdictions.

How to Use the Filters:

  • Age Groups: Filter the data to focus on specific age demographics
  • Detection Methods: Toggle between camera-issued and police-issued fines
  • Jurisdictions: Select specific states or territories to compare
  • Months: Focus on particular months or seasonal patterns

Hover over any cell to see detailed information about the number of fines issued. Use the "Reset Filters" button to clear all selections and return to the default view.

Key Insights from the Heatmap

  • NSW Dominates Fine Issuance: NSW (New South Wales) consistently issues the highest number of fines for mobile phone use while driving, as shown by the darkest shades across all months. This highlights NSW as the strictest enforcer among all jurisdictions.
  • QLD Lacks Age Group Data: Queensland (QLD) does not have any fines attributed to specific age groups; all fines are recorded under the "Unknown" age group. This suggests either a data collection gap or a reporting issue for age group information in QLD.
  • Police-Issued Fines Only in Some States: In ACT, NT, SA, and WA, all recorded fines are police-issued, with no camera-issued fines present in the data. This indicates that these jurisdictions rely solely on police enforcement for mobile phone use fines, unlike other states that utilize both detection methods.
