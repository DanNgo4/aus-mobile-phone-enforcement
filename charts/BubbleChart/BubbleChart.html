Bubble Chart: Fines vs Road Deaths

This chart illustrates the relationship between the total number of fines issued per jurisdiction and road deaths. The bubble size represents the number of fines per 10,000 licensed drivers in each jurisdiction, providing a normalized view of enforcement intensity relative to the driving population.

How to Read the Chart:

  • X-axis: Number of road deaths in each jurisdiction
  • Y-axis: Total number of fines issued
  • Bubble Size: Represents fines per 10,000 licensed drivers
  • Bubble Color: Different colors represent different jurisdictions

How to Use the Filters:

  • Age Groups: Filter the data to focus on specific age demographics
  • Detection Methods: Toggle between camera-issued and police-issued fines

Hover over any bubble to see detailed information about the jurisdiction, including road deaths, total fines, number of licenses, and fines per 10,000 drivers. Use the "Reset Filters" button to clear all selections and return to the default view.
